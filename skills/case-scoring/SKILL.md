# 案例评分排序 Skill

给每个参考案例打分并按参考价值排序，帮助快速找到最有参考价值的案例。

## 功能描述

- 多维度评分（距离、面积、时间）
- 加权总分计算（距离50% + 面积30% + 时间20%）
- 智能排序（总分 → 距离 → 时间）
- 自身拍卖案例单独列出（不参与评分）
- 评分明细可追溯

## 触发方式

### 自然语言触发
- "给这些案例打个分"
- "按参考价值排序"
- "哪个案例最有参考价值"
- "评分排序一下"

### 代码调用
```typescript
import { CaseScorer } from './case-scoring';

const scorer = new CaseScorer({
  propertyType: 'residential',
  targetArea: 100,
});

const result = scorer.scoreAndSort(cases, {
  selfAuctionItemIds: ['case1', 'case2'],
});

console.log('排序后案例数:', result.scoredCases.length);
console.log('最高分:', result.scoredCases[0]?.totalScore);
```

## 输入输出

### 输入
- `cases: ScoringCase[]` - 案例列表（过滤后的案例）
- `propertyType: string` - 资产类型
- `targetArea: number` - 抵押物建筑面积（平方米）
- `selfAuctionItemIds?: string[]` - 自身拍卖ID列表（单独列出）
- `sortPriority?: SortPriority[]` - 排序优先级，默认 ['totalScore', 'distance', 'time']

### 输出
- `scoredCases: ScoredCase[]` - 评分排序后的案例列表（不含自身拍卖）
- `selfAuctionCases: ScoringCase[]` - 自身拍卖案例（单独列出）
- `scoreBreakdown: ScoreBreakdown[]` - 评分明细
- `sortPriority: SortPriority[]` - 使用的排序优先级

## 评分规则

### 权重分配

| 维度 | 权重 | 满分 | 说明 |
|-----|------|------|------|
| 距离 | 50% | 50分 | 距离越近得分越高 |
| 面积 | 30% | 30分 | 面积越接近得分越高 |
| 时间 | 20% | 20分 | 时间越新得分越高 |
| **合计** | **100%** | **100分** | |

### 距离评分（50分）

| 距离范围 | 得分 | 说明 |
|---------|------|------|
| ≤ 500m | 50分 | 非常近，参考价值极高 |
| ≤ 1km | 45分 | 很近，参考价值很高 |
| ≤ 2km | 37.5分 | 较近，参考价值高 |
| ≤ 3km | 30分 | 适中，参考价值中等 |
| ≤ 5km | 15分 | 较远，参考价值一般 |
| > 5km | 5分 | 很远，参考价值较低 |

### 面积评分（30分）

面积差异率 = |案例面积 - 抵押物面积| / 抵押物面积 × 100%

| 差异率 | 得分 | 说明 |
|-------|------|------|
| ≤ 10% | 30分 | 非常接近，参考价值极高 |
| ≤ 20% | 25分 | 比较接近，参考价值很高 |
| ≤ 30% | 20分 | 适中，参考价值中等 |
| ≤ 50% | 12分 | 差异较大，参考价值一般 |
| > 50% | 5分 | 差异很大，参考价值较低 |

### 时间评分（20分）

| 时间范围 | 得分 | 说明 |
|---------|------|------|
| ≤ 90天 | 20分 | 非常新，参考价值极高 |
| ≤ 180天 | 17分 | 很新，参考价值很高 |
| ≤ 365天 | 13分 | 较新，参考价值高 |
| ≤ 540天 | 8分 | 适中，参考价值中等 |
| ≤ 730天 | 4分 | 较旧，参考价值一般 |
| > 730天 | 1分 | 很旧，参考价值较低 |

## 排序规则

### 主排序（按优先级从高到低）
1. **总分**（降序）- 总分越高越靠前
2. **距离**（升序）- 距离越近越靠前
3. **时间**（升序）- 时间越新越靠前

### 自身拍卖处理
- 自身拍卖案例不参与评分和排序
- 单独列在结果的 `selfAuctionCases` 字段中
- 使用方可以决定展示位置（如排在最前或最后）

## 特殊规则

- **无数据处理**：缺少距离/面积/时间数据的案例，对应维度得最低分
- **距离优先**：距离权重最高（50%），是最重要的参考因素
- **面积次之**：面积权重次之（30%），影响中等
- **时间辅助**：时间权重最低（20%），作为辅助参考
- **同分处理**：总分相同时，按距离排序；距离也相同时，按时间排序
