# 资产类型匹配 Skill

判断拍卖案例的资产类型（住宅/商业/工业/土地/特殊资产）及子类型，用于过滤和分类。

## 功能描述

- 识别拍卖案例的资产大类（住宅/商业/工业/土地/特殊资产）
- 识别资产子类型（商铺/写字楼/厂房/住宅用地等）
- 判断案例是否匹配目标类型
- 返回匹配置信度和匹配原因

## 触发方式

### 自然语言触发
- "判断这个是什么类型的房产"
- "这是住宅还是商业？"
- "类型匹配检查"
- "资产分类"

### 代码调用
```typescript
import { AssetTypeMatcher } from './asset-type-matcher';

const matcher = new AssetTypeMatcher();

const result = matcher.match({
  title: '佛山市禅城区滨海御庭住宅拍卖',
  description: '三室两厅，建筑面积120㎡',
  targetType: '住宅',
});

console.log('是否匹配:', result.isMatch);
console.log('匹配类型:', result.matchedType);
console.log('置信度:', result.confidence);
```

## 输入输出

### 输入
- `title: string` - 拍卖标题
- `description?: string` - 描述/详情（可选）
- `buildingArea?: number` - 建筑面积（可选，辅助判断）
- `targetType: string` - 目标资产大类（住宅/商业/工业/土地/特殊资产）
- `targetSubType?: string` - 目标子类型（可选）

### 输出
- `isMatch: boolean` - 是否匹配目标类型
- `matchedType: string` - 匹配到的大类
- `matchedSubType?: string` - 匹配到的子类型
- `matchReason: string` - 匹配原因（命中了哪个关键词）
- `confidence: number` - 匹配置信度（0-100）
- `matchedKeyword?: string` - 命中的关键词

## 类型映射规则

### 大类关键词

| 大类 | 匹配关键词 |
|-----|-----------|
| 住宅 | 住宅、公寓、别墅、小区、花园、家园、苑、府、邸、城、庭 |
| 商业 | 商业、商铺、门面、店铺、商场、购物中心、写字楼、办公、酒店、宾馆 |
| 工业 | 工业、厂房、仓库、厂区、工业园、产业园 |
| 土地 | 土地、用地、地块、宗地 |
| 特殊资产 | 采矿权、林权、海域使用权 |

| 大类 | 子类型 | 关键词 |
|-----|-------|--------|
| 商业 | 商铺 | 商铺、商业用房、门面 |
| 商业 | 商场 | 商场、购物中心、商业广场 |

### 特殊规则
- **住宅额外判断**：标题包含"室""厅""卧""号""栋"等住宅常用词，且有建筑面积，也判定为住宅
- **未匹配到**：标题为空时默认返回 isMatch=true（先保留，后面再过滤）
- **多类型命中**：取置信度最高的类型

## 匹配逻辑

```
1. 遍历各大类关键词
   └─ 检查标题中是否包含该类关键词

2. 遍历各子类型关键词（如果指定了子类型）
   └─ 检查标题中是否包含子类型关键词

3. 特殊判断（住宅）
   ├─ 有"室/厅/卧"等词 + 有面积 → 住宅
   └─ 有"栋/座/号楼"等词 + 有面积 → 住宅

4. 计算置信度
   ├─ 精准命中大类关键词 → 80-100分
   ├─ 命中子类型关键词 → 额外加10分
   └─ 辅助特征（面积、地址等）→ 额外加0-10分

5. 返回结果
   ├─ 匹配目标类型 → isMatch=true
   └─ 不匹配 → isMatch=false
```
